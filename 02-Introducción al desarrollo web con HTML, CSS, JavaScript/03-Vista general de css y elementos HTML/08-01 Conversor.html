Temperatura



Para convertir Celsius a Fahrenheit, usa esta fórmula reemplazando `C` con tu temperatura en grados Celsius: (C × 9/5) + 32
Peso



Para convertir kilogramos a libras, usa esta fórmula reemplazando `kg` con tu peso en kilogramos: kg x 2.205
Distancia



Para convertir kilómetros a millas, usa esta fórmula reemplazando `km` con tu distancia en kilómetros: km ÷ 1.609
