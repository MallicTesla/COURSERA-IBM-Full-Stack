Sistema Solar

El sistema solar está formado por el sol y todo lo que orbita alrededor del sol. Esto incluye los ocho planetas y sus lunas, los planetas enanos, los asteroides, los cometas y otros objetos pequeños.

Todos los planetas y los planetas enanos, los asteroides rocosos y los cuerpos helados del cinturón de Kuiper se mueven alrededor del Sol en órbitas elípticas en la misma dirección en la que gira el Sol. Este movimiento se denomina movimiento progrado o directo.
