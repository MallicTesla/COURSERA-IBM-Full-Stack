AJAX
"JavaScript y XML asincrónicos" que abarca más que llamadas al servidor asincrónicas a través de JavaScript y XML. No es un lenguaje o tecnología de programación, sino más bien
    un concepto de programación. Ajax representa una serie de técnicas que proporcionan aplicaciones web interactivas y más ricas a través de HTML, JavaScript, hojas de estilo
    en cascada y modificación de la página web a través del modelo de objetos de documento. Sin embargo, el nombre es engañoso porque hoy en día se usa comúnmente JSON en lugar
    de XML.

Clases
Las clases actúan como modelo o plantilla para construir objetos con características y comportamientos similares. Una clase encapsula datos (en forma de propiedades) y funciones
    (en forma de métodos) que trabajan con esos datos.

Script del lado del cliente
Un programa que acompaña a un documento HTML o está incrustado en HTML. Los scripts se ejecutan durante la carga de un documento o cuando se realiza una acción. Se pueden
    utilizar para validar formularios, procesar entradas o crear dinámicamente elementos de documentos. Incruste un script en HTML, con la <script>etiqueta de cualquiera de las
    siguientes maneras:     <script> // JS code </script>
                            <script src="path name"></script>
Utilice <noscript> la etiqueta para navegadores con JavaScript deshabilitado o aquellos que no admiten JavaScript.

Prototipos
Un prototipo de función le permite definir y agregar fácilmente propiedades o métodos a un objeto. Existen prototipos para todos los objetos que se pueden crear con la palabra
    clave "nuevo". Todos los constructores de objetos crean objetos que heredan propiedades y métodos definidos por el prototipo. En la creación de instancias, los objetos
    heredan el estado actual del prototipo. Sin embargo, tenga en cuenta que los scripts pueden anular las propiedades y funciones del prototipo. A continuación se muestra un
    ejemplo del uso de un prototipo para agregar un método a la clase Car:
function Car(make, model, year) {
this.make = make;
this.model = model;
this.year = year;
}
Car.prototype.getName = function() {
return this.make + ' ' + this.model + ' ' + this.year;
}
