"1. Creación de aplicaciones front-end enriquecidas con React y ES6
    React es una biblioteca de JavaScript eficiente y flexible para construir interfaces de usuario.

    Las nuevas características introducidas en JavaScript como parte de ES6 son let, const, arrow functions, promise y class.

    Las principales ventajas de utilizar JSX son que puede aprovechar toda la potencia de JavaScript en HTML y evitar aprender o utilizar un lenguaje de plantillas. Permite a
        React mostrar errores útiles y mensajes de advertencia.

    Los cuatro tipos de componentes React son Funcionales, de Clase, Puros y de Alto Orden.

    Los componentes funcionales son más útiles cuando no es necesario gestionar el ciclo de vida del componente. 

    Los componentes de clase son más versátiles.


---------------------------------------------------------------------------------------------------------------------------------
2. Componentes React
    State es un objeto JavaScript plano utilizado por React para representar información sobre la situación actual del componente.

    Props es una forma abreviada de propiedades, y se utilizan para pasar datos entre componentes React en un flujo unidireccional de padre a hijo.

    Puede pasar datos entre componentes de padre a hijo utilizando propiedades, de hijo a padre utilizando devoluciones de llamada, y entre hermanos.

    Los componentes se crean o se montan en el DOM; crecen actualizándose y luego mueren o se desmontan en el DOM. Esto se conoce como ciclo de vida de un componente.

    Los componentes React pueden probarse utilizando Mocha, Chai y Sinon, pero los enfoques preferidos son utilizar Jest y React Testing Library.


---------------------------------------------------------------------------------------------------------------------------------
3. React avanzado
    Los ganchos proporcionan una forma de utilizar funcionalidades como el contexto o el estado sin clases.

    Las entradas en React pueden ser de dos tipos: controladas o no controladas.

    Redux es una biblioteca de gestión de estado que se utiliza a menudo con React para manejar el estado de su aplicación.

    Los elementos Redux que intervienen en la actualización de las propiedades del componente son action, store y reducer.  

    Puede interactuar con datos asíncronos en su aplicación React Redux utilizando middleware.

    El flujo de datos en la aplicación React-Redux es unidireccional."
